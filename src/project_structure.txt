RISC-V Simulator Project Structure
==================================

Root Directory:
  - Launcher and test scripts
  - Documentation files

src/:
  - Core RISC-V implementation
  - Main GUI interface
  - Exception handling

src/UnitTests/:
  - Unit test files
  - Component-specific tests

examples/:
  - Sample assembly programs
  - Educational examples

docs/:
  - Documentation files
  - API references
